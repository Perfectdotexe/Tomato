# Tomato

### Free and multi-platform Software to use with the Pomodoro technique.

#### Platforms:
- Windows XP or higher
- GNU/Linux.
- MacOS
#### Requirements:
- JRE 7 or higher
- No Internet required
#### Language support:
- English
- Portuguese
- German
- French
- Spanish
- Klingon
